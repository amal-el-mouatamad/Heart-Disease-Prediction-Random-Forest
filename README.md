# ❤️ Heart Disease Prediction using Random Forest

## 📌 Project Overview

This project predicts whether a patient has heart disease using Machine Learning and the Heart Disease Dataset.

A Random Forest Classifier was trained after data preprocessing and evaluated using multiple classification metrics.

---

## 🎯 Objectives

- Explore the dataset
- Clean the data
- Remove duplicate records
- Analyze feature correlations
- Detect outliers
- Split the dataset into training and testing sets
- Train a Random Forest model
- Evaluate model performance

---

## 📊 Dataset Information

- Dataset: Heart Disease Dataset
- Number of samples: 302
- Number of features: 13
- Target:
  - 0 = No Heart Disease
  - 1 = Heart Disease

---

## 🛠 Technologies Used

- Scikit-learn
